# TP2

Idée Principale, Le joueur se retrouve, dans un bateau qui coule. Dans les 2 niveaux, ils doit ouvrir une porte avec une clef différente jusqu'à ce qu'il réeussice a sortir. Pour obtenir chaque clef, le joueur doit déplacer des objets sur des zones de détection et faire peut-etre du parcour. Le joueur doit se dépecher, car progressivement, l'eau va monter. une fois que l'eau à atteind sa hauteur maximal, un décompte commence et quand il se termine, le joueur va se noyer/perdre la partie. (quand il change de niveau, il est plus haut, dans le bateau, donc l'eau commence à ses pieds).

Level_1: 

obejectif principale, trouver clef pour ouvrir une porte.
obejectif secondaireLe, construir le raft.

concepte:
Le joueur doit pousser des boites de picèces de bateau,dans une zone de fabrication, pour fabriquer/spawn un mini raft/bateau de sauvetage, une fois fait une clef tombe d'une trape et en la récupérant, le joueur pourra ouvrir la porte.

level_2: 

objectif principale, trouver une clef pour ouvrir une porte.
objectif secondaire 1, envoyer signal d'alerte.
objectif secondaire 2, objectif secondaireLe, remplir le raft.

concepte:
Le joueur doit pousser des objet de survie Ex:(veste de sauvetage,telephone satellite,etc) dans une zone de détection(raft avec un pnj survivant). Une fois fait, une cleft tombe d'une trape et il peut la récupérer pour ouvrir la porte et en la récupérant, le joueur pourra ouvrir la porte.
